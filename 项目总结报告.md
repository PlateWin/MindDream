## 一、项目概况

**MindDream** 是一个面向心理学研究的梦境数据分析平台，集成梦境数据管理、多维度可视化统计与 AI 智能分析功能。项目由 17 人团队历时 6 个月完成，按 6 个阶段从需求分析推进至正式发布。

| 概要项 | 说明 |
|--------|------|
| 后端 | Spring Boot 3.2.1 + JPA + MyBatis Plus，15 个 REST API 端点 |
| 前端 | Vue 3.4 + Vite 5 + Element Plus + ECharts，Apple Liquid Glass UI |
| 数据库 | MySQL 8.0，dreams / statistics 两张核心表 |
| 团队 | PM×1 + 算法×3 + 前端×6 + 后端×6 + QA×1 |
| 代码规模 | 15 个 Java 类 + 7 个页面 + 9 个组件 + 9 套 CSS，约 5,500 行 |
| 许可证 | MIT |

本次总结围绕项目三大核心目标展开：**平台上线**、**项目开源**、**文档沉淀**。

## 二、目标一：平台上线

> **目标**：MindDream v1.0 完成生产环境部署，通过 PAT 验收测试，正式对外提供服务。

### 2.1 生产环境部署（后端 × 6）

| 任务 | 执行内容 | 状态 |
|------|----------|------|
| 数据库部署 | MySQL 8.0 安装 → 建库 `minddream` → 执行 `schema.sql` → JSONL 数据导入 | ✅ |
| 后端打包 | `mvn clean package -DskipTests`，生成 `minddream-backend-1.0.0.jar` | ✅ |
| 服务化部署 | 支持 `java -jar` 直接启动 + Linux systemd 服务化，JVM 参数调优（`-Xms512m -Xmx2g -XX:+UseG1GC`） | ✅ |
| Nginx 配置 | SPA 路由 fallback + `/api/` 反向代理 + 静态资源 1 年强缓存 | ✅ |
| 安全加固 | HTTPS（Let's Encrypt）、数据库权限最小化、UFW 防火墙、生产配置分离（`application-prod.yml` 不入库） | ✅ |
| 监控与备份 | 日志文件轮转配置 + MySQL 每日自动备份脚本（保留 30 天） | ✅ |

**部署架构**：

```
用户 → Nginx (:80/:443) → 前端静态文件 (dist/)
                         → /api/* 代理 → Spring Boot (:8080) → MySQL (:3306)
```

## 六、项目亮点与展望

### 6.1 项目亮点

1. **完整的前后端分离架构**：Spring Boot + Vue 3，Controller-Service-Repository 三层解耦，API 封装统一。
2. **Apple Liquid Glass 设计语言**：液态玻璃卡片、Morphing Blob 动态背景、3D 磁力倾斜、多主题切换，视觉效果出色。
3. **AI 五位一体分析能力**：综合解读 + 情感分析 + 符号解读 + 个人洞察 + AI 对话，覆盖梦境解读全场景。
4. **文档即产品**：12 份文档形成入门→开发→部署→接口→版本的完整闭环，新人可独立上手。
5. **开源友好**：MIT 许可证、清晰的项目结构、中英双语 README，降低社区参与门槛。

### 6.2 v2.0 规划

| 方向 | 计划项 |
|------|--------|
| 功能 | 用户系统（登录/注册）、梦境录入、个人梦境历史、关键词提取 |
| 算法 | BERT/RoBERTa 微调（6 分类目标 70%+）、Dify 工作流编排 |
| 前端 | 升级 React + Tailwind CSS + D3.js |

| 社区 | GitHub Wiki 完善、Issue/PR 模板、CONTRIBUTING.md、目标 50+ Stars |
