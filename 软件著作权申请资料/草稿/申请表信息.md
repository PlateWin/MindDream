➤面向领域 / 行业：心理健康与心理学数据分析
➤软件的主要功能：（1）梦境录入与管理：用户在首页仪表板填写姓名、性别、年龄段和梦境内容文本，提交后系统保存梦境记录并实时更新统计数据，支持录入后立即查看和管理全部梦境数据。（2）梦境列表浏览与筛选：以卡片网格形式分页展示全部梦境记录，每张卡片显示编号、性别标签、年龄段标签、梦境者姓名和内容摘要，支持按性别、年龄组和关键词进行实时筛选过滤。（3）梦境详情查看：展示单条梦境的完整报告文本、出现人物（中英文对照）和情感色彩标签，同时提供原始编码数据供研究者参考，页面设有AI分析快捷入口。（4）多条件高级搜索：提供独立的搜索页面，支持按性别、年龄组、关键词和时间段等多维度条件组合检索，搜索结果以卡片形式分页展示，支持页码跳转浏览。（5）多维度数据统计可视化：基于ECharts交互式图表展示梦境数据的性别分布环形图、年龄分布饼图、情感光谱柱状图和角色类型分布图，首页以Bento Grid布局汇总展示梦境总数、梦境者数、情感类型数和年龄组数等核心统计指标。（6）AI梦境智能分析：对选定梦境进行综合性AI分析，输出总体心理学解读摘要、情感分析（主要情感标签与强度百分比、次要情感标签组）、梦境符号识别与含义解读（附情境解释）、基于角色与情感生成的个性化洞察，以及改善睡眠与自我认知的行动建议。（7）AI梦境咨询对话：提供梦境主题的AI对话功能，用户可自由输入心理学相关疑问并获得AI顾问解答，支持一键快捷提问（预设常见梦境问题）、查看历史对话记录和消息时间戳。
➤技术特点：采用前后端分离架构，前端基于Vue 3 + Vite + Element Plus + ECharts构建，实现Apple Liquid Glass风格UI和3D交互效果；后端基于Spring Boot + JPA + MyBatis Plus构建REST API；使用MySQL存储结构化梦境数据；通过AI接口实现梦境智能分析和对话功能。
